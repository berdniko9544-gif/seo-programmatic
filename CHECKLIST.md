# ✅ CHECKLIST — Пошаговый план запуска

## 📋 Подготовка (один раз)

### Шаг 1: Установка зависимостей

- [ ] Node.js >= 18.x установлен
- [ ] NPM пакеты установлены (`npm install`)
- [ ] Vercel CLI установлен (`npm i -g vercel`)

**Проверка:**
```bash
npm run satellite:health
```

---

### Шаг 2: Получение API ключей

- [ ] Vercel токен получен ([https://vercel.com/account/tokens](https://vercel.com/account/tokens))
- [ ] Vercel Team ID получен (для Pro аккаунтов)
- [ ] (Опционально) Yandex Webmaster токен получен

**Настройка:**
```bash
export VERCEL_TOKEN=your_token_here
export VERCEL_TEAM=team_xxxxxxxxxxxxx  # для Pro
```

---

### Шаг 3: Проверка готовности

- [ ] Health check пройден без критических ошибок
- [ ] Все скрипты на месте (8 файлов в `scripts/`)
- [ ] Документация прочитана (`START_HERE.md`)

---

## 🔄 Миграция существующих сателлитов

### Шаг 1: Подготовка

- [ ] Создан бэкап существующих сателлитов

- [ ] Запущена миграция:

- [ ] Запущена сборка:

- [ ] Все сателлиты успешно собраны

### Шаг 4: Деплой

- [ ] Запущен деплой:

### Шаг 5: Проверка
