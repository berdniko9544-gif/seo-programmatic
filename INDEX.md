## 🛠️ Скрипты автоматизации

Все скрипты находятся в директории `scripts/`:

### 14. `satellite-generator.js` — Генератор одного сателлита
- Копирует базовый шаблон
- Генерирует уникальные данные
- Настраивает конфигурацию

### 15. `batch-generator.js` — Батч-генерация (20+ сателлитов)
- Параллельная генерация (5 одновременно)
- Поддержка разных ниш
- Логирование результатов

### 16. `migrate-satellite.js` — Миграция одного сателлита
- Создаёт бэкап
- Разбивает данные на модули
- Копирует новые компоненты

### 17. `migrate-all.js` — Батч-миграция
- Автоматическое обнаружение проектов
- Параллельная обработка
- Детальные логи

### 18. `build-all.js` — Сборка всех сателлитов
- Параллельная сборка (3 одновременно)
- Подсчёт страниц
- Статистика времени

### 19. `deploy-all.js` — Деплой на Vercel
- Автоматический деплой
- Поддержка Vercel Pro
- Сохранение списка URL

### 20. `submit-to-search.js` — SEO автоматизация
- Отправка в Google Search Console
- Отправка в Яндекс Вебмастер
- API интеграция

### 21. `master.js` — Главный оркестратор
- Управляет всем процессом
- Режимы: генерация, миграция, деплой
- Флаги для пропуска этапов

### 22. `health-check.js` — Проверка готовности
- Проверка Node.js и NPM
- Проверка Vercel CLI
- Проверка структуры проекта
- Проверка переменных окружения

## 📋 NPM скрипты

Все команды доступны через `npm run`:

```bash
# Генерация
npm run satellite:generate          # Один сателлит
npm run satellite:batch             # Батч-генерация

# Миграция
npm run satellite:migrate           # Один сателлит
npm run satellite:migrate-all       # Все сателлиты

# Сборка и деплой
npm run satellite:build-all         # Сборка всех
npm run satellite:deploy-all        # Деплой всех

# SEO
npm run satellite:submit-seo        # Отправка в поисковики

# Главный оркестратор
npm run satellite:master            # Полный цикл

# Проверка
npm run satellite:health            # Health check
```
