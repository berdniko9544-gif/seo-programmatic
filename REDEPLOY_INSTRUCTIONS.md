## Устранение проблем

### Ошибка: "No existing credentials found"
```bash
vercel login
```

### Ошибка: "Build failed"
Проверьте логи конкретного сателлита:
```bash
cd satellites/[satellite-name]
npm run build
```

### Ошибка: "Rate limit exceeded"
IndexNow имеет лимит. Подождите 1 час и повторите:
```bash
npm run network:submit-indexnow
```

## Мониторинг

### Еженедельная автоматизация
GitHub Actions автоматически запускает обновление контента каждое воскресенье в 3:00 МСК:
- Триггерит ISR (Incremental Static Regeneration)
- Отправляет в IndexNow
- Сохраняет логи (30 дней)

### Ручной триггер ISR
```bash
npm run network:refresh-content
```
