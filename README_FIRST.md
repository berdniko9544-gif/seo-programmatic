# 👋 НАЧНИТЕ ЗДЕСЬ

## Что это?

Полная система автоматизации для создания SEO-сателлитов промышленного масштаба.

**Возможности:**
- 🚀 Создание 20+ сателлитов за 60-90 минут
- 🔄 Миграция существующих сателлитов на новую архитектуру
- ☁️ Автоматический деплой на Vercel
- 🔍 Автоматическая отправка в Google/Яндекс

---

## ⚡ Быстрый старт (3 команды)

### У вас уже есть сателлиты на Vercel Pro?

```bash
# 1. Проверка готовности
npm run satellite:health

# 2. Миграция (замените путь на свой)
npm run satellite:migrate-all -- --dir /path/to/your/satellites

# 3. Деплой
npm run satellite:build-all && npm run satellite:deploy-all
```

### Хотите создать новые сателлиты?

```bash
# Создать 5 сателлитов для теста
npm run satellite:master -- --generate 5 --niche crypto --pages 500
```
