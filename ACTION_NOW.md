## 🚀 ЧТО ДЕЛАТЬ СЕЙЧАС

### Шаг 1: Открыть GitHub Actions (1 минута)

**Ссылка для копирования:**
```
https://github.com/berdniko9544-gif/seo-programmatic/actions/workflows/vercel-daily-generation.yml
```

1. Скопировать ссылку выше
2. Открыть в браузере
3. Войти в GitHub (если не вошли)

### Шаг 2: Запустить workflow (30 секунд)

На странице GitHub Actions:

1. Найти кнопку **"Run workflow"** (справа вверху, серая кнопка)
2. Нажать на неё
3. Откроется форма с параметрами:
   - **Use workflow from**: `Branch: main` (оставить как есть)
   - **Satellites to generate**: `5` (для первого теста)
   - **Pages per satellite**: `1000` (оставить как есть)
4. Нажать зеленую кнопку **"Run workflow"**

### Шаг 3: Ждать завершения (15-20 минут)

Workflow автоматически:
- ✅ Скачает последний код (с улучшениями)
- ✅ Сгенерирует 5 новых сателлитов
- ✅ Соберет их (build)
- ✅ Развернет на Vercel
- ✅ Отправит sitemap в Yandex

**Прогресс можно смотреть:**
- На той же странице GitHub Actions появится новый запуск
- Нажать на него, чтобы видеть логи в реальном времени

### Шаг 4: Проверить результаты (5 минут)

После завершения workflow:

1. **Скачать URL новых сателлитов:**
   - На странице завершенного workflow
   - Прокрутить вниз до секции "Artifacts"
   - Скачать `vercel-urls` (zip файл)
   - Распаковать и открыть `vercel-urls.txt`

2. **Проверить 2-3 сателлита:**
   - Открыть любой URL из файла
   - Проверить canonical теги (Ctrl+U → найти "canonical")
   - Проверить image sitemap: добавить `/image-sitemap.xml` к URL
   - Нажать любую CTA кнопку → должен перенаправить на https://1mb3-guide-2026.vercel.app/

## ❓ Если что-то пошло не так

### Workflow не запускается:
- Проверить, что вы на странице `vercel-daily-generation.yml`
- Проверить, что выбрана ветка `main`
- Попробовать обновить страницу

### Workflow завершился с ошибкой:
- Открыть логи workflow
- Найти красную строку с ошибкой
- Проверить GitHub Secrets (Settings → Secrets → Actions):
  - DEEPSEEK_API_KEY
  - VERCEL_TOKEN
  - VERCEL_ORG_ID

### Сателлиты не развернулись:
- Проверить Vercel Dashboard
- Проверить лимиты Vercel Pro
- Попробовать запустить с 3 сателлитами вместо 5
