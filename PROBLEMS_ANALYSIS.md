# Анализ критических проблем сателлитов

## Проблема 1: Кнопки не работают ❌

**Причина:** В `.env.local` файлах сателлитов указаны старые Cloudflare Workers URL вместо реальных Vercel URL.

**Пример:**
```
NEXT_PUBLIC_SITE_URL=https://seo-sat-ai-copywriting-1773660281654-0.berdniko9544.workers.dev
```

**Должно быть:**
```
NEXT_PUBLIC_SITE_URL=https://ai-copywriting-1773660281654-0.vercel.app
```

**Последствия:**
- Внутренние ссылки ведут на несуществующие домены
- Кросс-линки между сателлитами не работают
- Кнопки перехода на главный сайт могут не работать

---

## Проблема 2: Одинаковые шаблоны ❌

**Статус:** Частично решено, но недостаточно

**Факты:**
- Шаблоны РАЗНЫЕ: `atlas`, `signal`, `workshop`, `fieldbook`, `navigator`
- Но визуальные различия минимальны

**Текущее распределение:**
- ai-copywriting-0: atlas
- ai-design-1: signal  
- ai-automation-2: signal
- ai-copywriting-8: atlas

**Проблема:** Шаблоны отличаются только цветами и мелкими деталями, структура идентична.

---

## Проблема 3: Одинаковый контент ❌❌❌ КРИТИЧНО

**Причина:** Генератор контента НЕ использует `satelliteNumber` для вариации.

**Код проблемы:**
```javascript
generateArticles(blueprint, count = 18) {
  for (let index = 0; index < count; index++) {
    const topic = topics[index % topics.length];
    articles.push({
      title: `${topic} в 2026`,  // ← ОДИНАКОВО для всех сателлитов ниши!
      h1: topic,
      desc: `${topic}. Разбор спроса...`
    });
  }
}
```

**Результат:**
- `ai-copywriting-1773660281654-0` и `ai-copywriting-1773660748483-0` имеют **ИДЕНТИЧНЫЕ** статьи
- Все 4 сателлита ai-copywriting имеют одинаковый контент
- Это дубликат контента → Google Duplicate Content Penalty

**Пример дубликатов:**
```
Сателлит 1: "Как продавать AI-копирайтинг малому бизнесу в 2026"
Сателлит 2: "Как продавать AI-копирайтинг малому бизнесу в 2026"  ← ДУБЛИКАТ!
```

---

## Решения

### 1. Исправить URL (СРОЧНО)
- Обновить все `.env.local` с правильными Vercel URL
- Передеплоить все 28 сателлитов

### 2. Добавить вариацию контента (КРИТИЧНО)
- Использовать `satelliteNumber` в генерации
- Добавить вариации заголовков:
  - Сателлит 1: "Как продавать AI-копирайтинг малому бизнесу"
  - Сателлит 2: "AI-копирайтинг для малого бизнеса: гайд 2026"
  - Сателлит 3: "Продажа услуг AI-копирайтинга: пошаговая инструкция"
- Менять структуру описаний
- Варьировать ключевые слова

### 3. Усилить визуальные различия шаблонов
- Разные layouts (sidebar vs no-sidebar)
- Разные hero sections
- Разные navigation styles
