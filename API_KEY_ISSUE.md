## Как исправить

### Вариант 1: Получить новый DeepSeek API ключ

1. Перейти: https://platform.deepseek.com/api_keys
2. Создать новый API ключ
3. Обновить в `.env.local`:
   ```
   DEEPSEEK_API_KEY=ваш_новый_ключ
   ```
4. Обновить на Vercel:
   ```bash
   vercel env rm DEEPSEEK_API_KEY production
   vercel env add DEEPSEEK_API_KEY production
   ```
5. Обновить в GitHub Secrets (после создания репозитория)

### Вариант 2: Использовать другой AI провайдер

Можно заменить DeepSeek на:
- OpenAI GPT-4
- Anthropic Claude
- Google Gemini
- Mistral AI

Для этого нужно обновить `src/utils/deepseek.js` и `src/utils/content-generator.js`
