## Следующие Шаги

### 1. Передеплой Сателлитов (Обязательно)
Чтобы активировать кросс-ссылки, нужно передеплоить все сателлиты:

```bash
npm run satellite:vercel-deploy
```

**Время:** ~15 минут (28 сателлитов × 0.5 минуты)

### 2. Первая Отправка в IndexNow
После деплоя отправьте все URL в IndexNow:

```bash
npm run network:submit-indexnow
```

**Важно:** Загрузите сгенерированный файл `[API-KEY].txt` в корень главного сайта.

### 3. Добавить REVALIDATE_SECRET в GitHub Secrets
Для работы автоматического обновления контента:

1. Сгенерируйте секретный ключ: `openssl rand -base64 32`
2. Добавьте в GitHub Secrets как `REVALIDATE_SECRET`
3. Добавьте в `.env.local` всех сателлитов

### 4. Деплой Главного Сайта
Задеплойте главный сайт со страницей `/projects`:

```bash
vercel --prod
```
