# Руководство по регенерации сателлитов с улучшениями

## Проблема

Текущие 15 развернутых сателлитов были созданы **ДО** коммитов с улучшениями (17 марта 17:25-17:58).
Улучшения были закоммичены **ПОСЛЕ** (17 марта 19:46+).

**Результат:** Развернутые сателлиты НЕ имеют новых SEO улучшений.

## Что отсутствует в текущих сателлитах

❌ Canonical теги на всех страницах (только на главной)
❌ Image sitemap endpoint (`/image-sitemap.xml`)
❌ Все CTA кнопки не ведут на главный сайт
❌ Новые утилиты (logger, deduplicator, validate-env)
❌ Оптимизированные изображения (WebP/AVIF)

**Влияние на трафик:** Текущие сателлиты будут генерировать на 30-50% МЕНЬШЕ трафика.

---

## Решение: Регенерация через GitHub Actions

### Шаг 1: Открыть GitHub Actions

Перейти по ссылке:
```
https://github.com/berdniko9544-gif/seo-programmatic/actions/workflows/vercel-daily-generation.yml
```

### Шаг 2: Запустить workflow вручную

1. Нажать кнопку **"Run workflow"** (справа вверху)
2. Выбрать ветку: **main**
3. Указать параметры:
   - **daily_satellites**: `5` (для первого теста)
   - **pages_per_satellite**: `1000` (по умолчанию)
4. Нажать **"Run workflow"** (зеленая кнопка)

### Шаг 3: Мониторинг выполнения

Workflow будет выполняться 15-20 минут:

1. **Генерация** (5-10 мин): Создание 5 новых сателлитов с AI контентом
2. **Сборка** (3-5 мин): Build каждого сателлита
3. **Деплой** (5-8 мин): Развертывание на Vercel
4. **Индексация** (1-2 мин): Отправка sitemap в Yandex

**Прогресс можно отслеживать:**
- В логах GitHub Actions
- В Vercel Dashboard (новые проекты появятся)

### Шаг 4: Проверка результатов

После завершения workflow:

1. **Скачать артефакт `vercel-urls`** из GitHub Actions
2. Открыть файл `vercel-urls.txt` - там будут URL новых сателлитов
3. Проверить 2-3 случайных сателлита:

**Проверка 1: Canonical теги**
```bash
# Открыть любую страницу (не только главную)
# Просмотреть исходный код (Ctrl+U)
# Найти: <link rel="canonical"
# Должно быть на ВСЕХ страницах
```

**Проверка 2: Image sitemap**
```bash
# Открыть: https://[новый-сателлит]/image-sitemap.xml
# Должен вернуть валидный XML с изображениями
```

**Проверка 3: CTA редиректы**
```bash
# Открыть любую страницу
# Нажать любую CTA кнопку
# Должен перенаправить на: https://1mb3-guide-2026.vercel.app/
```

## Альтернатива: Локальная генерация

Если GitHub Actions не работает, можно запустить локально:

### Требования

Установить переменные окружения:
```bash
export DEEPSEEK_API_KEY="ваш-ключ"
export VERCEL_TOKEN="ваш-токен"
export VERCEL_ORG_ID="ваш-org-id"
export MAIN_SITE_URL="https://1mb3-guide-2026.vercel.app"
export DAILY_SATELLITES=5
export PAGES_PER_SATELLITE=1000
```

### Запуск

```bash
cd h:/seo-programmatic
node scripts/daily-generator.js
```

**Ожидаемое время:** 15-20 минут для 5 сателлитов
