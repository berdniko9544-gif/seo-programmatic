## 🚀 ОСТАЛОСЬ: Запустить один файл (2 минуты)

### Шаг 1: Откройте командную строку
```
Win + R → cmd → Enter
```

### Шаг 2: Перейдите в папку проекта
```
cd D:\seo-programmatic
```

### Шаг 3: Запустите скрипт
```
SETUP_GITHUB.bat
```

### Что произойдет:
1. Скрипт попросит авторизоваться в GitHub (откроется браузер)
2. Создаст репозиторий seo-programmatic
3. Запушит весь код (25 коммитов)
4. Добавит все секреты автоматически
5. Включит GitHub Actions

**Всё автоматически!**

## 📞 Если что-то пойдет не так

### Ошибка: "gh: command not found"
Установите GitHub CLI: https://cli.github.com/

### Ошибка: "Repository already exists"
Это нормально, скрипт просто запушит код в существующий репозиторий

### Ошибка: "Authentication failed"
Запустите вручную: `gh auth login` и следуйте инструкциям
