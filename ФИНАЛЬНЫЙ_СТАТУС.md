## ⚠️ Важно: DeepSeek API ключ

**Текущий ключ невалиден** (401 Authentication Fails)

Получить новый ключ:
1. https://platform.deepseek.com/api_keys
2. Создать новый API ключ
3. Обновить на Vercel:
   ```bash
   vercel env rm DEEPSEEK_API_KEY production
   vercel env add DEEPSEEK_API_KEY production
   ```
4. Обновить в GitHub Secrets

**Без валидного ключа:**
- ✅ Статические страницы работают
- ✅ ISR обновления работают
- ❌ AI генерация нового контента не работает
