# 🔑 Создание GitHub Personal Access Token

## Шаг 1: Откройте страницу создания токена

В вашем браузере перейдите по ссылке:
https://github.com/settings/tokens/new

## Шаг 2: Заполните форму

**Note (название токена):**
```
seo-programmatic-deploy
```

**Expiration (срок действия):**
```
90 days (или No expiration)
```

**Select scopes (выберите права):**
Отметьте галочками:
- ✅ **repo** (полный доступ к репозиториям)
- ✅ **workflow** (доступ к GitHub Actions)
- ✅ **admin:org** → **read:org** (чтение организаций)

## Шаг 3: Создайте токен

1. Прокрутите вниз
2. Нажмите зеленую кнопку **"Generate token"**
3. **ВАЖНО:** Скопируйте токен СРАЗУ (он начинается с `ghp_`)
4. Токен показывается только один раз!

## Шаг 4: Отправьте мне токен

Скопируйте токен и отправьте мне в чат.

Формат токена: `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
