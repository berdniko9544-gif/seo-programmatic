### Шаг 1: Запушить код на GitHub

**Вариант A: Автоматический (рекомендуется)**
```bash
cd D:\seo-programmatic

# Windows
setup-github.bat

# Linux/Mac
bash setup-github.sh
```

**Вариант B: Вручную**
```bash
# 1. Авторизация
gh auth login

# 2. Создание репозитория и push
gh repo create seo-programmatic --public --source=. --remote=origin --push

# 3. Добавление секретов
gh secret set DEEPSEEK_API_KEY --body "sk-0745998fd651417cb649a864d5f427de"
gh secret set VERCEL_TOKEN --body "vca_4GmyixM5cqJicGNspgfCDWaILBrZA8Hz0RM9P0TuybzW3ernof1sBzB7"
gh secret set VERCEL_TEAM --body "berdniko9544-2708s-projects"
gh secret set REVALIDATE_SECRET --body "your_random_secret_here_change_this"

# 4. Включение workflows
gh workflow enable daily-generation.yml
gh workflow enable isr-update.yml
```

**Вариант C: Через веб-интерфейс**
1. Создать репозиторий: https://github.com/new
2. Имя: `seo-programmatic`
3. Public
4. Не инициализировать с README
5. Запушить:
   ```bash
   git remote remove github
   git remote add origin https://github.com/YOUR_USERNAME/seo-programmatic.git
   git push -u origin main
   ```
6. Добавить секреты: Settings → Secrets and variables → Actions
7. Включить Actions: Actions → Enable workflows

### Шаг 2: Получить новый DeepSeek API ключ

⚠️ **Текущий ключ невалиден** (401 Authentication Fails)

1. Перейти: https://platform.deepseek.com/api_keys
2. Создать новый API ключ
3. Обновить на Vercel:
   ```bash
   vercel env rm DEEPSEEK_API_KEY production
   vercel env add DEEPSEEK_API_KEY production
   ```
4. Обновить в GitHub:
   ```bash
   gh secret set DEEPSEEK_API_KEY
   ```
